# phaser3-npm-webpack

A base for building a Phaser 3 application in node / webpack land. Write ES6 games.

- Phaser 3
- Babel
- Express
- Pug
- Browsersync


## Installation
```
npm install
```

## Running
```
npm start
```

## Browser
```
localhost:3000   phaser app
localhost:3001   
localhost:3002   Browser Sync admin UI
```
